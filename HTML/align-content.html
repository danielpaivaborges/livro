ALIGN-CONTENT

ALIGN CONTENT alinha as linhas do container em relação ao eixo vertical. Esta propriedade só funciona se existir mais de uma linha de flex-itens, para isso o flex-wrap precisa ser wrap, além disso o efeito apenas será visualizado caso o container seja maior que a soma das linhas dos itens. Isso significa que se não definirmos height para o container a propriedade não influencia no layout.
